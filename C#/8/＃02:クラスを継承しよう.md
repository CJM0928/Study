## C# ↔ Java 상속(Inheritance) 비교 — Box / JewelryBox 예제

### 핵심 개념
- Box = 부모 클래스
- JewelryBox = Box를 상속한 자식 클래스
- 자식 클래스는 **부모 생성자(base/super)**를 호출해야 한다
- 자식은 부모 메서드 사용 가능, + 자기만의 메서드 추가 가능

---

### C# 코드
```
using System;

class Lesson08
{   
    public static void Main()
    {
        var box = new Box("鋼鉄の剣");
        box.Open();
        
        var jewelryBox = new JewelryBox("魔法の指輪");
        jewelryBox.Look();  // 자식 클래스 고유 메서드
        jewelryBox.Open();  // 부모 클래스 메서드 상속됨
    }
}

class Box 
{
    public string Item { get; private set; }
    
    public Box(string item)
    {
        Item = item;
    }
    
    public void Open()
    {
        Console.WriteLine("宝箱を開いた。" + Item + "を手に入れた。");
    }
}

class JewelryBox : Box
{
    public JewelryBox(string item) : base(item)
    {
    }
    
    public void Look()
    {
        Console.WriteLine("宝石はキラキラと輝いている。");
    }
}

```
### 실행 결과
```
宝箱を開いた。鋼鉄の剣を手に入れた。
宝石はキラキラと輝いている。
宝箱を開いた。魔法の指輪を手に入れた。
```

### Java 버전 (동일 구조)
```
class Box {
    private String item;

    public Box(String item) {
        this.item = item;
    }

    public void open() {
        System.out.println("宝箱を開いた。" + item + "を手に入れた。");
    }
}

class JewelryBox extends Box {
    public JewelryBox(String item) {
        super(item);  // C#의 base(item)
    }

    public void look() {
        System.out.println("宝石はキラキラと輝いている。");
    }
}

public class Lesson08 {
    public static void main(String[] args) {
        Box box = new Box("鋼鉄の剣");
        box.open();

        JewelryBox jewelryBox = new JewelryBox("魔法の指輪");
        jewelryBox.look();
        jewelryBox.open();
    }
}

```

### 실행 결과 (C#과 100% 동일)
```
宝箱を開いた。鋼鉄の剣を手に入れた。
宝石はキラキラと輝いている。
宝箱を開いた。魔法の指輪を手に入れた。
```

### C# ↔ Java 상세 비교표

| 관점            | C#                           | Java                    | 해설                   |
| ------------- | ---------------------------- | ----------------------- | -------------------- |
| 상속 문법         | `class A : B`                | `class A extends B`     | 기호만 다름               |
| 부모 생성자 호출     | `base(item)`                 | `super(item)`           | 반드시 생성자 첫 줄          |
| 필드 private 접근 | `private set`으로 setter 제한 가능 | getter/setter 직접 만들어야 함 | Java는 프로퍼티 문법 없음     |
| 문자열 출력        | `$"{Item}"` 가능               | `String` + 연산           | C#의 문자열 인터폴레이션이 더 간결 |
| 객체 생성 시 타입 추론 | `var obj = new Box(...);`    | `var`는 Java10+ 가능       | C# var가 더 일찍 등장      |
| 메서드 호출        | `box.Open()`                 | `box.open()`            | C#은 PascalCase 관례    |
