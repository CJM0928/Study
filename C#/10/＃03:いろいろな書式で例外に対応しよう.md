## いろいろな書式で例外に対応しよう (C# Exception 출력 방식)

### C# 코드 요약 (해설 포함)

```
using System;

class Lesson10
{
    public static void Main()
    {
        Console.WriteLine("Hello World");
        
        try
        {
            int number = 0;
            int answer = 100 / number;  // 예외 발생
            Console.WriteLine(answer);
        }
        catch (Exception e)
        {
            // 사용자 메시지
            Console.WriteLine("0では割り算できません");

            // 예외 전체 정보 출력
            Console.WriteLine(e);

            // 예외 메시지만 출력 (주석처리됨)
            // Console.WriteLine(e.Message);
        }
        finally
        {
            Console.WriteLine("Hello C#");
        }
    }
}
```

---

### 실행 결과 예시

```
Hello World
0では割り算できません
System.DivideByZeroException: Attempted to divide by zero.
   at Lesson10.Main()
Hello C#
```

- e 을 Console.WriteLine에 넣으면 예외 타입 + 메시지 + 스택트레이스 전체가 출력됨
- e.Message 는 짧게 "Attempted to divide by zero." 만 출력

---

### C# Exception 출력 방식 비교

| 출력 방식                          | 예시              | 내용            |
| ------------------------------ | --------------- | ------------- |
| `Console.WriteLine(e)`         | 전체 예외 정보        | 타입 + 메시지 + 스택 |
| `Console.WriteLine(e.Message)` | 메시지만            | 간단한 오류 이유     |
| 커스텀 메시지                        | `"0では割り算できません"` | 사용자 친화적 메시지   |
