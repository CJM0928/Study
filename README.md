# 1. 코드 블록

```언어 로 열고, 반드시 ```로 닫기

출력 예시는 ```text 또는 그냥 ``` 사용

코드 블록 안에는 #, -, * 같은 Markdown 문법이 적용되지 않음

---

# 2. 제목(Heading)

# ~ ###### 까지 사용 가능 (#이 많아질수록 소제목)

제목 뒤에는 반드시 공백 하나 넣기

❌ #제목

✅ # 제목

---

# 3. 목록(List)

순서 있는 목록: 1., 2., 3.

순서 없는 목록: -, *, +

목록 안에 코드 블록 넣을 땐 빈 줄 한 줄 추가해야 함

---

# 4. 구분선

--- 또는 *** 사용

반드시 한 줄 전체에만 써야 함

❌ --- 내용

✅ ---

# 9. 인라인 주석/설명

코드 블록 안에 주석은 언어 주석 문법 사용 (//, #, <!-- ... -->)

코드 블록 밖에서는 > 인용문 사용 가능

예:

여기서 중요한 점은...

📌 흔한 실수

코드 블록 닫는 ```를 빼먹음 → 이후 내용이 전부 코드처럼 나옴

제목에 공백 없음 (#제목) → 그냥 일반 텍스트로 출력됨

출력 예시를 코드 블록으로 안 감쌈 → GitHub UI 버튼이 이상하게 붙음

---를 다른 텍스트 옆에 씀 → 구분선 대신 "—" 문자로 표시됨
